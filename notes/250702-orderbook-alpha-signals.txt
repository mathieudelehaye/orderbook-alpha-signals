Your notebook performs a comprehensive alpha signal backtesting workflow for
  orderbook-based trading strategies:

Main functions:
1. Data loading - Loads 1-minute AAPL OHLCV data (1,585 rows from Feb 5-7, 2025)
2. Signal engineering - Creates 5 alpha signals: VWAP deviation, mean reversion,
momentum, RSI, volume surge
3. Individual backtesting - Tests each signal separately with transaction costs
4. Combined strategy - Weights signals by Sharpe ratios and combines them
5. Risk analysis - Analyzes drawdowns, returns distribution, rolling metrics
6. Cost sensitivity - Tests performance across different transaction cost levels
7. Performance summary - Compares all strategies and provides insights

Current results show:
- Dataset: 1,585 minute bars from real AAPL data
- Combined strategy achieved better risk-adjusted returns than individual
signals
- Transaction cost analysis helps determine strategy viability
- Comprehensive performance metrics and visualizations

The notebook is fully functional and demonstrates a professional quant analysis
workflow.
